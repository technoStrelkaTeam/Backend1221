2. Add new interpreter -> add local interpreter...
3. Выберите Virtualenv Environment. Укажите путь (лучше всего Calendarium/backend/venv) и нажмите OK. Потом дождитесь создания venv

### II. Установите зависимости
1. В терминале пропишите команды ниже
2. ```bash
   pip install -r requirements.txt
   ```

### III. Установите OLLAMA для встроенной в проект нейросети и запустите его
1. Скачайте Ollama с официального сайта https://ollama.com/download , либо в терминале:
   
   На Windows: ```irm https://ollama.com/install.ps1 | iex```
   
   На Linux или macOS: ```curl -fsSL https://ollama.com/install.sh | sh```
3. В терминале выполните команды
   
   ```bash
   ollama serve
   #(если выдает ошибку, то ollama уже запущена, пропускайте ollama serve)
   
   ollama pull gemma3:1b


P.S. Если вам позволяют ресурсы компьютера, то можете взять более хорошую модель, либо вовсе зарегестрироваться на Ollama и взять облачную. Тогда в ollama pull вместо gemma3:1b пишите любую другую модель из https://ollama.com/search и в файле Calendarium/backend/api/config.py в константе AI_MODEL заменяете имя модели на свою.
P.S.S. Предупреждаем, gemma3:1b отвечает очень и очень плохо, лучше берите более мощную модель

### IV. Запустите проект
1. В терминале выполните команду

   ```bash
   uvicorn api.main:app --reload  --host 0.0.0.0
   ```
